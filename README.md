DropdownLoginJquery
===================

 

Steps to run this application
-----------------------------

[Download and install node js current version and
npm](https://nodejs.org/en/download/current/)

Install gulp globally

`$ npm install --global gulp-cli`

`$ npm install`

From inside the folder run

`$ gulp`
